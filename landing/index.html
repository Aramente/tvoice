Tvoice

Mobile-first terminal for steering AI coding agents from your phone. One command, a QR code, and your phone becomes the command center for Claude Code.

npx tvoice

Scan the QR on your phone. You're in.

*

Special key toolbar

ESC, TAB, CTRL, ALT, arrows — all 44px touch targets. Sticky modifiers. Swipe up for symbols. AI row auto-appears when Claude Code is detected.

~

tmux persistence

Each tab is a tmux session. Disconnect, lock the screen, switch networks — reconnect and pick up where you left off.

>

Local Whisper voice

Tap mic, speak, text lands in your terminal. whisper.cpp runs on your Mac — audio never leaves the machine. Works inside iOS PWAs where Web Speech is blocked.

#

Zero-config tunnel

Cloudflare Quick Tunnel or Tailscale Serve. One flag, auto-HTTPS, QR code in your terminal. No port forwarding, no DNS, no nginx.

@

iOS text selection

Long-press a word, drag the handles to extend, tap Copy. Same gesture you use in Notes — adapted for a terminal canvas.

!

Push notifications

Command done? Agent needs input? Get a push on your phone. Works on iOS 16.4+ after installing to the home screen.

$

2FA + audit log

Optional TOTP second factor. Device-bound cookies. JWT rotation. Append-only JSONL audit log. Refuses to run as root. Security headers on every response.

%

OLED dark theme

#0A0A0A background (no OLED smearing), #E0E0E0 text (no halation). 5 built-in themes + custom JSON import. Pinch to zoom.

How it works

Phone (PWA)                         Mac (your laptop)
  xterm.js                            Express + ws
  + toolbar                           + node-pty -> tmux
  + voice recorder     <-- wss -->    + whisper.cpp
  + push client                       + JWT auth
                                      + Cloudflare Tunnel / Tailscale

Tvoice is self-hosted software, not a cloud service. You run it on your own Mac. Your phone connects to it. Your credentials (Claude, GitHub, SSH, AWS) stay on your machine — Tvoice is a pipe to your existing terminal, not a custodian of your secrets.

Quick start

npx tvoice                       # Cloudflare tunnel (default)
npx tvoice --tunnel tailscale    # private tailnet
npx tvoice --no-tunnel           # localhost only

Requirements: Node 18+. Optional: tmux (session persistence), cloudflared or tailscale (remote access), whisper-cpp + ffmpeg (voice input).

brew install tmux cloudflared ffmpeg whisper-cpp

Security model

Tvoice gives the authenticated user a live shell on the machine running the server. That is the maximum privilege a process can have on your account. Read SECURITY.md before deploying outside your own laptop.

  • One-time login tokens (15 min, single use)
  • Device-bound JWT cookies with configurable TTL
  • Optional TOTP second factor
  • JWT secret rotation (--rotate-secret)
  • Per-IP rate limiting on login + transcribe
  • Append-only audit log
  • CSP, Referrer-Policy, X-Frame-Options on every response
  • Refuses to run as root or bind to 0.0.0.0 without --allow-lan
